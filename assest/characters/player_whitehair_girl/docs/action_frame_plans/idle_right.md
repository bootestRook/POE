# Action Frame Plan：idle_right

## 基础信息

- action_id：`idle_right`
- character_id：`player_whitehair_girl`
- direction：`right`
- frame_count：4
- frame_size：256×256
- loop_type：loop
- view：2.5D / dimetric 轻俯视
- anchor_point：脚底中心固定
- 生成规则：只生成 `right` 母版；`left` 后续必须由 `right` 水平镜像得到

## 角色不变量

- 白色几何长发结构不变：前额大块刘海、右侧垂发片、后方大块长发都必须保留。
- 服装不变：棕色披肩、棕色上衣、米色内衬/长条布料、深色长裤、棕色短靴。
- 配色不变：白发、浅蓝眼、浅肤色、棕/米/深灰黑主色。
- 比例不变：Q 版头身比例，头大身短，简洁扁平卡通描线。
- 朝向不变：始终面向屏幕右侧。
- 不新增武器、饰品、护甲、背包、特效、文字、水印、场景。

## 禁止项

- 不做走路、跑步、攻击、施法、跳跃或转身。
- 不改变角色发型、服装、配色、比例、画风、视角。
- 不让脚底锚点滑动。
- 不生成 left。
- 不出现文字、水印、网格、边框、场景或 UI。

## 验收标准

- 4 帧均为 256×256 PNG。
- 背景透明；如不透明，必须可后处理去背。
- 角色仍明显是同一个白发女孩。
- idle 动作幅度很小，只表现呼吸和披肩/头发轻摆。
- 首尾能自然循环。
- 脚底锚点稳定。
