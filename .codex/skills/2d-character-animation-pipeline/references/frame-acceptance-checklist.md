# 序列帧验收清单

## 格式

- [ ] 每帧尺寸正确。
- [ ] PNG 格式。
- [ ] 有 alpha 通道。
- [ ] 背景透明。
- [ ] 无文字。
- [ ] 无水印。
- [ ] 无边框。
- [ ] 无网格线。

## 一致性

- [ ] 脸一致。
- [ ] 发型一致。
- [ ] 服装一致。
- [ ] 武器一致。
- [ ] 身体比例一致。
- [ ] 主色一致。

## 动作

- [ ] idle 只有轻微呼吸，不滑步。
- [ ] walk 左右脚交替清晰。
- [ ] walk 首尾可循环。
- [ ] run 有前倾、压缩、冲刺感。
- [ ] attack 有预备、出手、命中、收招。
- [ ] cast 有聚能、释放、回位。
- [ ] hit 受击方向清晰。
- [ ] death 最终帧静止。

## 方向

- [ ] right 面向屏幕右侧。
- [ ] left 由 right 镜像生成。
- [ ] left/right 帧数一致。
- [ ] left/right 动作节奏一致。

## 锚点

- [ ] 脚底参考线稳定。
- [ ] 角色中心没有异常漂移。
- [ ] 攻击和施法允许身体前倾，但支撑点不能乱跳。
